Minesweeper with Javascript Language

Posted on 30th April 2018

Javascript Language

Remind the definition - What does JavaScript mean? Javascript (JS) is a scripting languages, primarily used on the Web. It is used to enhance HTML pages and is commonly found embedded in HTML code. JavaScript is an interpreted language. Thus, it doesn't need to be compiled. JavaScript renders web pages in an interactive and dynamic fashion. This allowing the pages to react to events, exhibit special effects, accept variable text, validate data, create cookies, detect a user’s browser, etc.

In the JavaScript language, describe all the use cases for each of below symbol.

Parentheses ()
Parentheses can be used for various purposes in Javascript. When you need to group several syntax in Javascript you can use it. Thus, you can organaise complex syntax by order which is grouped.
e.g. 'var x = (4+y-z) / 5'
Also, in function, we can show parameters by using parentheses. Normally, when the fuction is called, it can forward values by parameters in the codes. Not only it is used to show parameters, but the parentheseses are also used to define and call a fuction itself.
e.g. 'setZipCode (streetNo, cityName), getData ()'
Brackets
Arrays are lists of things. they can be lists of values. properties in Objects. When we define the arrays like below
e.g. var tmpArray = new Array(10) Its array with ten elements has indexs from 0 to 9, and we can refer to a Specific item in an array by placing the index inside the braackets after array names. e.g. tmpArray[0] = "valude-1"
Also, when you declare the array, you can add elements to the array from using brackets
e.g. var laptop = ["samsung", "LG", "SONY"]
Braces
Braces are used to define properties in Objects. You need to know about the objects is that they have properties. Although there are a number of ways to create objects, but we should stick to the best one, and we can use objects by literal with braces.
e.g. var laptop = { modelNo: "Sens700", price: 500, color: "white"};
Single quotes & Double quotes
In Javascript, both single quotes and double quotes are used to define and show string values like below, and they are interchangable.
var tmpStr = "messages"
However, when you need to use both single and double quotes as string values, you should use back-slash symbol. e.g. console.log ('I\'am strings.'); or console.log ("This is a \"A\" grade.");
