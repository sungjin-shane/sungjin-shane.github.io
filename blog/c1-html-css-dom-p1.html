What is the box model?

The box model is our toolkit for customizing layout scheme, and it consists of four elements.
A major part of our work as a web developer will be to apply rules from the CSS box model to turn a design mockup into a web page. why we have to learn all these rules instead of just uploading a giant static image of a web page (i.e., a mockup) to a web server and calling it a day.
1.Content : The text, image, or other media content in the element.
2.Padding : Padding property defines the padding for the selected element.
3.Boarder : A line drawn around the content and padding of an element

What is meant by boxifying design?

Although it sounds strange, we can find a answer if we imagine most display screen in devices such as laptop's monitor, tablets and mobile phones.

This is because boxifying design is the most suitable solution to print all web elements on the screen in rectangular shape. All web elements like image, text and linked objects will be easily placed on web pages. After that, it can be modified from applying various attributes on the elements such as position, size and colour etc. Also, this design methodology allows people to establish standard framework and hierarchy.
