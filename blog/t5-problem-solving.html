Explain what functions are and why they are useful.

I think the fuction is a magic box which is operated independentely and it can be reusable.

When we use the fuction, we are able to make clear codes and reduce the same source codes which has the same purpose. The function is designed to operate independentely, thus, developers are able to call the fucntions in sources codes where the function is needed.
If we make a e-Commerce web site there will be web pages which need login-process. In this cases, can you imagine to make the same source codes for login process in each web pages
It will be really inefficient and leads to negative side effect. This is beacise that if we have to modify the login process we must update every web pages that have login process. However, if we make login process by function we can easily call the function in the every web pages which are required the login process.
